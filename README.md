<h1>Sakura Minecraft Launcher</h1>
<h3>樱花启动器 —— 简约、方便、快捷的启动器</h3>
欢迎使用 Sakura Minecraft Launcher<br>
一款使用 PHP 开发的 Minecraft 启动器，支持原版/Forge，读 json 方式启动<br>
支持正版登录，纯命令行，无图形界面，理论上支持 Linux<br>
所需的 PHP 环境需要安装 pthreads 组件才能运行本启动器。<br>
目前已经测试可以启动的版本：1.7.10 1.8.8 1.10.2 1.12.2<br>
已测试可以启动的 Forge：1.7.2 1.7.10 1.8.8<br>
并且还在不断更新中，如果发现任何问题欢迎在 Issues 里提交。<br>
